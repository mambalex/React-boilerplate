# React Boilerplate

## Features

1. Typescript
2. Redux
3. Saga
4. Jest
5. Enzyme
6. Prettier / Eslint / Airbnb config
7. Import alias
8. Sass
9. Bootstrap
10. Axios
11. Classnames
12. Typesafe-actions
13. Husky, which can prevent bad git commit, git push
14. SEO friendly, uses `react-helmat` for head tags management

## Quick start

`npm start`
